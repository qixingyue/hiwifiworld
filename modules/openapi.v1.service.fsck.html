get_state (opt)
获取任务状态. 获取任务的状态

Parameters:

  • opt 可选参数

Returns:

    {code, msg, data}

Usage:

     fsck.stat 说明
     fask_start   - 正在扫描,   (同一时间 只能扫一个设备)
     fsck_finish  - 扫描完毕,   (扫描操作已结束, 有扫描记录可以查询)
     fsck_not_run - 没有扫描任务(没有扫描任务, 也没有扫描记录)

 ret = get_state()
 //返回值:
 {
   "pid": 805,  //进程id, <0 说明未扫描
   "ret": 1,    //扫描结果, <0 说明未扫描完毕 (或 未扫描)
   "stat": "fsck_finish",                 //扫描状态
   "part_path": "\/dev\/mmcblk0",         //扫描的设备
   "args": "-v -C5 -y -f -- callback -m ",//扫描参数
   "pass": 5,   //扫描进度 0~5 个阶段
   "cur": 462,  //当前进度 当前阶段 0~max 个刻度
   "max": 462,
 }

check (opt)
扫描指定设备

Parameters:

  • opt 扫描选项,必需

Returns:

    {code, msg, data}

See also:

Usage:

     //调用示例:
 opt={
       device: '/dev/mmcblk0p2', //要扫描的分区, 必需
      verbose: false, //true|false; 是否获得详细的 report
        level: 1,     //1-4; 扫描深度
     callback: 'n'    //'m' - mount, 'u' - umount, 'n' - nothing; 扫描完毕后的动作
 }
 或
 opt={
         device: '/dev/mmcblk0p2'
    custom_args: '-v -f -y -- callback -m', //扫描参数 opt_verbose + opt_level + opt_callback
 }
 ret=check(opt)

 //关于扫描深度:
   1: -f -n       // check not change
   2: -f -y       // check & repair
   3: -f -y -c    // check & repair & mark_badblock by read only
   4: -f -y -c -c // check & repair & mark_badblock by non-destructive read-write

 //返回值:
 如果已有任务在执行, 返回 0x6020014, 否则 code 0
