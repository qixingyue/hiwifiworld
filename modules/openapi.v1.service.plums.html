status (args)
查询插件状态

Parameters:

  • args {app_name}

Returns:

    {code, msg, data}

Usage:

     有些插件可能无状态

 args={
  app_name="sample"     // 插件英文名
 }

 data={
  status_str="{\"status\":\"running\"}"       //  插件status方法原始信息
  status_obj={        //  插件status方法对象化，通常里面包含status属性
    status="running"
  }
 }
