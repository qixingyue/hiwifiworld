Module openapi.v1.service.sd_backup

SD卡数据备份: 将卡挂载为只读, 让用户通过 samba 访问(备份)数据

Info:

  • Copyright: (c) 2016 HiWiFi Elite Co., Ltd.
  • Author: hao.chen

Functions

get_status () 获取 sd_backup 状态.
start_backup () 启动 sd_backup 功能.
stop_backup () 停止 sd_backup 功能.


Functions

get_status ()
获取 sd_backup 状态.

Usage:

     //返回值
{
    "status": "stopped",  //未启动, running 已启动
    "is_old_part": "0",   //不是旧分区, 不支持 start_backup
    "share_path": ""      //共享项路径, 启动后 可以通过 win资源管理器 访问
}
start_backup ()
启动 sd_backup 功能.
只读挂载 旧分区格式的 sd卡, 通过 samba 共享data分区
调用此接口前 先通过 get_status 方法 确认 是 旧分区格式的sd卡
异步执行, 调用后 通过 get_status 获取 状态
stop_backup ()
停止 sd_backup 功能.
卸载卡data分区, 移除 samba 共享项
