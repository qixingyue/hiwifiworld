get_realspeed (args)
实时速度

Parameters:

  • args {mac}

Returns:

    Table {code, msg, data}

Usage:

     args={
  mac="00:71:CC:4D:D8:9A"    // 设备mac，可选。mac为空表示获取所有终端实时速度以及路由器出口总速度(包括本地收发包).
 }
 data={
  "details":[
    {
      "mac":"00:71:CC:4D:D8:9A",    //终端mac
      "download_speed":26,          // 下行速度，bit/s
      "upload_speed":790            // 上行速度，bit/s
    },
    {
     "mac":"D4:1E:07:27:58:85",
     "download_speed":0,
     "upload_speed":0
    }
  ],
  "total":{                         // 总速度
    "upload_speed":1023,            // 上行速度，bit/s
    "download_speed":144            // 下行速度，bit/s
   }
 }
