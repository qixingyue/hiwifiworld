get_total ()
获取Qos总配置

Usage:

     data = {
    "mode":1|2|3          //Qos工作模式 1:智能 2:优先级 3:限速
    "total_up":1000    //上行总带宽 单位:bps
    "total_down":2000  //下行总带宽 单位:bps
    "active" true|false   //Qos是否开启标识
    "total_devices":23    //Qos配置的总设备数
    "used_devices":12     //Qos配置的活动设备数
    "guest_up":50         //访客网络上行限速值 单位:bps
    "guest_down":60       //访客网络下行限速值 单位:bps
    "ip_range":[          //Qos例外ip列表
    "192.168.1.0/24",
    "192.168.3.10",
    "192.168.5.10-192.168.5.20"
    ]
  }
list (args)
获取终端qos配置

Parameters:

  • args {}

Usage:

     args={
  "mac": "D4:EE:07:18:CF:33",   // 终端mac,获取该终端的qos配置(无参数则获取所有终端的配置)
 }

 data={
   "mac": "D4:EE:07:18:CF:33",   // 终端mac
   "ip": "192.168.199.100",   // 终端ip
   "mode":1|2|3                  //终端工作模式
   "prio":1|2|3                  //终端优先级
   "up": 50,                     // 上行限速值 单位:bps
   "down": 100,                  // 下行限速值 单位:bps
   "update_time":1446107665,     // 最后设置日期
   "online":1|0,     // 终端在线状态  1:在线 0:不在线
   "type":"station|ap|iot|guest",     //终端类型
   "working":1|0            //终端Qos是否运行  1:运行 0:未运行
 }
set (args)
配置终端qos

Parameters:

  • args {}

Usage:

     args={
  "mac":"38:48:4C:9D:8B:F9"  //终端mac
  "mode":"1|2|3|4|5",        //工作模式,1:智能;2:优先级;3:限速;4:一键加速;5：保证带宽模式
  "prio":1|2|3               //终端优先级
  "up": 50,                  //上行限速值 单位:bps
  "down": 100,               //下行限速值 单位:bps
  "stop_timeout":100         //终端限速超时时间 单位：分钟
 }
