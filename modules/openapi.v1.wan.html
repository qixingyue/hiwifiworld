get_pppoe_status (args)
获取pppoe拨号状态

Parameters:

  • args 返回值

Usage:

     ret = get_pppoe_status(args)
 data = {
    status=0|-1|9999,
    remote_message="E=691 auth error",
    remote_err_code="691",
    remote_err_msg="auth error",
    special_dial=1,
    special_dial_num=4,
    special_dial_max=11,
    mac_clone=0|1,
    mc_max=0|3,
    mc_index=1|2,
    mc_macaddr="ab:cd:ef:12:34:56",
    smart_pppoe={
       smart=1,
       start_time=1438156945,
       step_start_time=1438156976,
       step_name="100half",
       step_map=["normal","100half","10full","10half","normal"]
    }
 }
 status  0 为正常  -1 为连接中, 其他数字为不正常
 remote_message 拨号信息，E=691代表691错误
 remote_err_code 拨号错误码
 remote_err_msg 拨号错误信息

 smart_pppoe 智能拨号信息
 step_name 正在进行哪哪种尝试
 step_start_time 这种拨号尝试开始时间
 start_time 智能拨号流程开始时间
 step_map 依次尝试的拨号步骤

 special_dial 是否特殊拨号
 special_dial_num 特殊拨号到第几步
 special_dial_max 特殊拨号总数量

 mac_clone 是否mac克隆，1克隆
 mc_max 总共需要克隆几个MAC
 mc_index 当前已经克隆到第几个MAC
 mc_macaddr 当前正在使用的克隆mac
get_status (args)
获取wan信息

Parameters:

  • args {}

Returns:

    Table {code, msg, data}

See also:

Usage:

     data={
   wan_type="dhcp",   // 或者是 pppoe 拨号，static 静态ip，wisp 无线中继
   up=true|false,      // true为wan口已启动
   dev_link=true|false,  // true为wan口插了网线
   pending=true|false,   // true为拨号中
   uptime=120,           // wan启动时间，单位秒
   wan_ip="202.102.188.1",     // wan口ip
   wan_mac="d4:ee:07:11:22:33",// wan口mac
   wan_mask="255.255.255.0",   // 子网掩码
   ipv6_address=[              // ipv6
     {"mask":64,"address":"4006:e024:680:4cf8::1"}
   ],
   dns=["114.114.114.114", "8.8.8.8"], // DNS
   gateway="192.168.10.254",   // 网关
   inet_status=1|0,  // wan口是否联通（与上层连接正常，但不保证互联网畅通）
   uptime=1112,      // wan口启动时长，单位秒
   duplex="full",     // full全双工， half半双工
   speed="100M",      // 速率
   switch_status_wan="100full",   // 100M全双工, 10half=10M半双工, auto=自动选择
   inactive={                     // 未生效的属性
     "dns-server":["192.168.5.7","192.168.5.17"],    // 默认获得的dns，但因为自定义了dns，未生效
     "route":{},
     "dns-search":["hiwifi-inc.net"],
     "ipv4-address":{},
     "ipv6-address":{}
   }
 }

 具体配置信息和拨号错误信息
