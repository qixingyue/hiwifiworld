get_phylong_state (args)
获取长网线patch状态

Parameters:

  • args {}

Returns:

    Table {code, msg, data}

Usage:

     ret = get_phylong_state(args)

 data = {
  support=1,        // 1为支持长网线，0为不支持
  enable=1          //
 }

 enable
 默认不启动长网线patch
 默认状态为-1
 手动关闭状态为0
 手动开启状态为1
 云平台推送开启的状态为2
 只有状态为1或2时，才自动开启长网线patch
