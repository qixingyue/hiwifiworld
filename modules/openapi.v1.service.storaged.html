Functions

list ()
存储列表

Usage:

     disk.status 说明
     ready        - 挂载成功, 设备就绪
     mount_partly - 部分分区未挂载成功, (如果设备支持格式化, 可以格式化修复, 数据会丢失; 如果不支持格式化, 可以将就用)
     mount_fail   - 所有分区挂载失败, (可以格式化修复)
     umounted     - 卸载成功
     fsck         - 正在进行磁盘扫描, 详情查询 fsck.get_stat 接口 ( fsck状态 不能进行 mount,umount,format )

 支持 mount  的状态有: umounted, mount_partly, mount_fail (后两种 相当于 失败重试)
 支持 umount 的状态有: ready,    mount_partly, mount_fail
 支持 format 的状态有: ready,    mount_partly, mount_fail, umounted (所有稳态)
