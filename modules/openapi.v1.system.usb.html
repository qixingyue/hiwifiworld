Functions

get_info ()
获取usb状态

Returns:

    Table {app_code, app_msg, app_data}

Usage:

     data= {
    speed_priority_support=0, // 是否支持切换usb速率优先，1为支持，默认0不支持
    speed_mode="2",           // 当前速率。3表示3.0，2表示2.0
    speed_priority="off"      // 速率优先状态。默认off，表示如果对2.4G无线有干扰，会自动降为2.0。
                                 on为优先保证usb速率，忽略对2.4G无线的干扰.
 }
